Создай полноценное одностраничное веб-приложение на React для управления проектами и пайплайнами. Приложение должно служить дашбордом, где пользователь может видеть все свои проекты и управлять процессами внутри них.

Ключевые требования:

Дизайн и UX:

Стиль должен быть чистым, профессиональным и минималистичным, в духе современных SaaS-инструментов, таких как Linear, Jira или Asana.

Используй темную тему.

Интерфейс должен быть полностью адаптивным и отлично выглядеть на всех устройствах.

Используй качественные SVG-иконки для всех элементов управления.

Структура и страницы:

Главная страница (Дашборд проектов):

Отображает сетку или список карточек проектов.

Каждая карточка должна показывать название проекта, его краткое описание и ключевую статистику (например, "Активных пайплайнов: 3").

Должна быть яркая кнопка "+ Создать проект".

Страница проекта:

Пользователь попадает на нее, кликнув на карточку проекта.

На странице отображается детальная информация о проекте.

Основной элемент — список или таблица всех пайплайнов, принадлежащих этому проекту. Для каждого пайплайна нужно показать его название, статус ("Активен" / "Неактивен") и время последнего запуска.

Должна быть кнопка "+ Создать пайплайн".

Данные:

Так как это прототип, используй статические mock-данные прямо в коде для имитации ответа от API. Данные должны быть структурированными и реалистичными, включая несколько проектов и по 2-3 пайплайна в каждом.

Технологии:

Используй React с функциональными компонентами и хуками.

Для стилизации используй Tailwind CSS.

Весь код должен быть в одном файле.
